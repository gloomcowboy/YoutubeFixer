# YoutubeFixer 
Youtube was getting way too much of my attention latelly ...

## What it does

- Automatically redirects the home page to your subscriptions feed
- Hides the Shorts shelf
- Hides recommended/related videos sidebar
- Hides the "Up Next" autoplay widget

## Instructions

- **Place the 3 files inside a folder**
- **In your Browser (must be chromium based) go to the extensions page**
- **Toggle Developer Mode**
- **Click on "Load Unpacked"**
- **Select the folder with the 3 files.**

## Files

manifes.json -> Tells the browser what extension is and where to run
block.css -> Hides the desired page elements
content.js -> Watches the page  / redirects the home page to your subscriptions feed
